# MENUMAT — Menumaestro

## O projekte
Restaurant menu generator — generovanie denneho menu, export na socialne siete a marketing.

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (Postgres + Auth + Storage + Edge Functions)
- Google Gemini API
- Bun (package manager)
- Lovable (UI builder, sync cez GitHub)
