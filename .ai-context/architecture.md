# Architektura — MENUMAT

## Overview
Restaurant menu generator s AI. Generovanie denneho menu, export sablon na socialne siete.

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (auth, DB, storage, edge functions)
- Google Gemini API
- Bun

## Struktura projektu
src/           — React app
supabase/      — Supabase config, migrations, edge functions
public/        — staticke assety
.lovable/      — Lovable konfig
